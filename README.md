## Setup Instructions

### Prerequisites
Install `uv`:
- Mac/Linux: `curl -LsSf https://astral.sh/uv/install.sh | sh`
- Windows: `powershell -c "irm https://astral.sh/uv/install.ps1 | iex"`

### Installation
```bash
git clone <repo-url>
cd face-and-digit-classification
uv sync
```

### Run
```bash
uv run python src/main.py
```
